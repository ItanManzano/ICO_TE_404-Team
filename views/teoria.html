Ingeniería de Software

La Ingeniería de Software (I.S) es una disciplina que se encarga del diseño, desarrollo, prueba, mantenimiento y gestión de sistemas de software de forma sistemática, estructurada y eficiente.

Su objetivo principal es crear software de alta calidad, cumpliendo con los requisitos del cliente, dentro de los plazos y presupuesto establecidos.

Incluye actividades como:

  • Análisis de requisitos
  • Diseño del sistema
  • Programación
  • Pruebas (testing)
  • Documentación
  • Mantenimiento

Ciclo de Vida del Desarrollo de Software

El Ciclo de Vida del Desarrollo de Software (SDLC, por sus siglas en inglés) es el proceso estructurado que se sigue para crear un sistema de software, desde que nace la idea hasta que deja de usarse.

Fases típicas del ciclo de vida:

  • Análisis: Estudiar la viabilidad del proyecto y definir cómo funcionará el sistema.
  • Diseño: Crear la arquitectura del sistema y sus componentes.
  • PImplementación (codificación): Programar el software.
  • Pruebas: Verificar que todo funcione correctamente.
  • Despliegue: Instalar el software para que los usuarios lo utilicen.
  • Mantenimiento: Corregir errores, actualizar o mejorar el sistema.
